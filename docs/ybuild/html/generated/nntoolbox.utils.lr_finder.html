https://arxiv.org/pdf/1506.01186.pdf

find_lr(lr0: float = 1e-07, lr_final: float = 10.0, warmup: int = 15, beta: float = 0.98, verbose: bool = True, display: bool = True, callbacks: Optional[List[Callback]] = None) → Tuple[float, float][source]¶

Start from a very low initial learning rate, then gradually increases it up to a big lr until loss blows up :param lr0: intitial learning rate :param lr_final: final (max) learning rate :param warmup: how many iterations to warmup :param beta: smoothing coefficient for loss :param verbose: whether to print out the progress :param display: whether to graph :param callbacks: an optional list of callbacks to process input :return: a base_lr and the best lr (base_lr = best_lr / 4)
