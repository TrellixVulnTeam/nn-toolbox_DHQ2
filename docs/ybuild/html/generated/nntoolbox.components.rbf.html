References:

Parameters

  • in_features – dimension of input

  • out_features – number of centers

  • trainable_centers – whether the center can be moved

  • normalized – whether the output should be normalized (i.e sum to 1)

  • kernel – (optional) a distance-based kernel function

  • initial_centers – (optional) initial centers placement
